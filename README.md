# **Shipment Service Front**

> :alembic: This application provide interface for managing shipments.


## :construction: Install
```sh
npm install
```

## :rocket: Usage
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
